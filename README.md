## Что делает

- добавляет меню `График фотографов -> Создать месяц`
- создаёт новый лист месяца копированием предыдущего шаблонного листа
- запрашивает месяц, год, нормы часов и отпуск по каждому активному сотруднику
- распределяет рабочие дни, выходные и дежурства по правилам из листа `Настройки`
- гарантирует, что в каждый день месяца хотя бы один фотограф находится в смене
- не допускает серии длиннее `3` обычных выходных подряд у одного сотрудника
- копирует стили обычной смены, выходного, отпуска и дежурства из шаблона и служебных образцов
- после генерации показывает отчёт: дежурства по сотрудникам, отпуска и итог проверки правил
- поддерживает запуск через меню и через ручную кнопку-рисунок на листе `Настройки`

## Как устроен сценарий

1. Пользователь запускает генерацию из меню или ручной кнопкой на листе `Настройки`.
2. Скрипт берёт предыдущий месячный лист как шаблон.
3. С листа `Настройки` читаются постоянные правила сотрудников.
4. Из диалога берутся месяц, нормы часов и отпуск.
5. Решатель подбирает допустимый график:
   учитывает фиксированные выходные, четверги, пару `сб+вс`, дежурства, покрытие каждого дня и лимит `не больше 3 выходных подряд`.
6. Создаётся новый лист месяца и показывается отчёт по проверкам.

## Как использовать

1. Создайте Apps Script-проект, привязанный к рабочей Google-таблице.
2. Перенесите в него файлы проекта.
3. Откройте таблицу и дождитесь появления меню `График фотографов`.
4. Проверьте служебный лист `Настройки`.
5. Укажите для сотрудников постоянные правила:
   `Фиксированные выходные`
   `Требуется 1 пара сб+вс`
   `Все четверги рабочие`
   `Участвует в дежурствах`
6. Запустите `График фотографов -> Создать месяц` или создайте ручную кнопку-рисунок на листе `Настройки` и привяжите к ней `showCreateMonthDialog`.

## Ручная кнопка на листе `Настройки`

1. Откройте лист `Настройки`.
2. Нажмите `Вставка -> Рисунок -> Создать`.
3. Нарисуйте прямоугольник и подпишите его `Создать новый месяц`.
4. Сохраните рисунок и разместите его справа от настроек.
5. Нажмите на рисунок, затем на меню `⋮`.
6. Выберите `Назначить скрипт`.
7. Введите имя функции `showCreateMonthDialog`.

Проверка:
- при нажатии на кнопку должно открываться окно генерации
- если кнопка не сработала, резервный способ запуска — меню `График фотографов -> Создать месяц`

## Важные допущения v1

- рабочая смена фиксированная: `8` часов
- норма часов должна быть кратна `8`
- поддерживается только один тип исключения: отпуск
- отпуск можно задавать диапазоном через границу месяцев; в график попадут только дни создаваемого месяца
- обычные выходные `OFF` не могут идти серией длиннее `3`; отпуск в этот лимит не входит
- дежурство помечается той же цифрой `8`, но отдельным стилем
- дежурный назначается каждый день, включая выходные
- равномерность дежурств считается по будням; выходные назначения балансируются отдельно и не ломают будничный баланс
- источник состава сотрудников — предыдущий месячный лист
- программное создание кнопки на листе отключено; ручная кнопка в таблице считается основным вариантом

## Замечание по стилям

Лист `Настройки` хранит служебные образцы форматов. Это позволяет не терять оформление отпуска и дежурства, даже если в предыдущем месяце таких ячеек не было.
