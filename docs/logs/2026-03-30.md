- Добавлено пользовательское меню в Google Sheets для запуска генерации нового месяца.
- Реализован HTML-диалог для выбора месяца, года, норм часов и отпусков.
- Добавлен служебный лист `Настройки` с постоянными правилами сотрудников.
- Реализован генератор месячного листа с копированием предыдущего шаблона.
- Добавлен алгоритм распределения рабочих дней, обязательных четвергов, фиксированных выходных, пары `сб+вс` и будничных дежурств.
- Добавлено копирование и сохранение образцов форматов для обычной смены, выходного, отпуска и дежурства.
- После первой проверки на живой таблице уточнено правило покрытия: теперь в любой день месяца должен быть хотя бы один сотрудник в смене.
- После первой проверки на живой таблице уточнено правило дежурств: дежурный ставится каждый день, а равномерность считается по будням.
- Добавлена автоматическая пост-проверка после генерации с отчётом по дежурствам, отпускам и выполнению правил.
- Добавлена строгая проверка отпуска: диапазон должен целиком относиться к создаваемому месяцу, иначе генерация завершается ошибкой.
- После уточнения пользовательского сценария строгая проверка отпуска смягчена: диапазон может пересекать соседние месяцы, а генератор берёт только дни целевого месяца.
- Добавлена кнопка-ячейка на лист `Настройки`, открывающая диалог генерации при нажатии на неё; реализация упрощена до одной ячейки для большей надёжности.
- Для удобной ручной вставки в Apps Script подготовлен отдельный файл `ready-to-paste/Code.gs` с актуальным кодом и временной функцией принудительного создания кнопки.
- После серии ошибок Apps Script программное создание кнопки на листе удалено; переход выполнен на ручную кнопку-рисунок, привязанную к `showCreateMonthDialog`.
- Добавлена отдельная инструкция `ready-to-paste/MANUAL_BUTTON_SETUP.md` для ручной настройки кнопки в Google Sheets.

## Важные решения

- Цвета не захардкожены: скрипт берёт форматы из шаблонного листа и сохраняет образцы на листе `Настройки`.
- Дежурство остаётся значением `8`, а визуально определяется отдельным стилем.
- Для выходных дней дежурства тоже назначаются, но их распределение не влияет на баланс будничных дежурств.
- Меню `График фотографов` остаётся официальным резервным способом запуска на случай проблем с ручной кнопкой.
- При конфликте имени новый месячный лист создаётся с суффиксом, без перезаписи существующего.
- Если ограничения несовместимы, генерация останавливается с ошибкой, а не создаёт частично корректный график.

## Что ещё нужно сделать в живой таблице

- Перенести файлы проекта в привязанный Apps Script.
- Заполнить постоянные правила сотрудников на листе `Настройки`.
- Проверить генерацию на реальных данных марта и следующего месяца, затем скорректировать настройки сотрудников при необходимости.
